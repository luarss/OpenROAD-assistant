# OpenROAD-assistant

Objective is to build a LLM Chatbot that is able to answer user queries for OpenROAD.

## Quickstart

```
# clone current repo 
git clone https://github.com/luarss/OpenROAD-assistant.git
git checkout colab standalone

# standalone running 
!python chat.py 

```
